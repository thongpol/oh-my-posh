install-shells

info

If you have no idea which shell you're currently using, Oh My Posh has a utility switch that can you tell you (not relevant for the Powershell module).

oh-my-posh --print-shell
  • powershell
  • zsh
  • bash
  • fish
  • nu

Edit $PROFILE in your preferred PowerShell version and add the following line.

oh-my-posh --init --shell pwsh --config ~/jandedobbeleer.omp.json | Invoke-Expression

Once added, reload your profile for the changes to take effect.

. $PROFILE

Add the following to ~/.zshrc:

eval "$(oh-my-posh --init --shell zsh --config ~/jandedobbeleer.omp.json)"

Once added, reload your profile for the changes to take effect.

source ~/.zshrc

Add the following to ~/.bashrc (or ~/.profile on MacOS):

eval "$(oh-my-posh --init --shell bash --config ~/jandedobbeleer.omp.json)"

Once added, reload your profile for the changes to take effect.

. ~/.bashrc

Or, when using ~/.profile.

. ~/.profile
caution

It's advised to be on the latest version of fish. Versions below 3.1.2 have issues displaying the prompt.

Initialize Oh My Posh in ~/.config/fish/config.fish:

oh-my-posh --init --shell fish --config ~/jandedobbeleer.omp.json | source

Once added, reload your config for the changes to take effect.

. ~/.config/fish/config.fish

Set the prompt and restart nu shell:

Nu < 0.32.0#

config set prompt "= `{{$(oh-my-posh --config ~/jandedobbeleer.omp.json | str collect)}}`"

Nu >= 0.32.0#

config set prompt "(oh-my-posh --config ~/jandedobbeleer.omp.json | str collect)"

Restart nu shell for the changes to take effect.
